Install as a Python package
---------------------------

Install the latest **released version** from the Python Package Index
`PyPI <https://pypi.org/project/sphinxawesome-theme/>`_:

.. literalinclude:: ../install/includes/install.sh
   :language: bash

To install the latest **development version**, run:

.. code-block:: bash

   pip install git+https://github.com/kai687/sphinxawesome-theme.git

After installing the theme, you can :doc:`add it to your project <../add/index>`.
